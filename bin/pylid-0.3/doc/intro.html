Introduction

PyLid is an integrated unit-test and coverage analysis tool.

Features

  • Statistical coverage analysis.
  • Source annotation for a detailed view of coverage data.
  • Fine-grained selection of unit-tests from the command line. Allows you to conveniently run all unit tests in a directory, all tests in a given file, all tests in a given test-suite, or a specific test in a given test-suite.

Requirements

PyLid requires Python 2.3 or later.
