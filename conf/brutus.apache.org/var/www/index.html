Apache Gump on Brutus

Welcome to Brutus, the machine that the Apache Software Foundation dedicates to Gump.

The following is available:

  • Main Gump run[uses Sun's 1.4.x JVM and Gump code from 'live' branch] (runs at 0:00,3:00,9:00,15:00,21:00 pacific time)
  • Run that uses Sun's JVM 1.5.x (runs at 6:00 pacific time)
  • Run that uses Sun's JVM 1.4.x but gump's code from trunk (runs at 12:00 pacific time)
  • Run that uses the free Kaffe VM (runs at 18:00 pacific time)

Enjoy.
